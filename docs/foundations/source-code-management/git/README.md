# Git Fundamentals

Git is a distributed version control system used to track and manage source code changes.

## Core Concepts

- Repository, commit, branch, merge
- Rebase vs merge strategy
- Remote tracking and synchronization
- Conflict resolution

## Essential Commands

```bash
git status
git add .
git commit -m "message"
git pull --rebase
git push
git log --oneline --graph
```

## Best Practices

- Commit small logical changes.
- Rebase local branches before merge.
- Avoid force-push on shared branches.
