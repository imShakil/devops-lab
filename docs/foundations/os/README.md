# Operating Systems Fundamentals

Operating system knowledge is essential for troubleshooting, performance tuning, and secure infrastructure operations.

## Topics

- `linux/` Linux command line, processes, filesystems, and permissions
- `networking/` TCP/IP, DNS, routing, and troubleshooting basics
- `security/` Hardening, authentication, and access control

## Core Skills

- Process and service management
- File and permission management
- System resource monitoring
- Network diagnostics
- Basic host security practices
