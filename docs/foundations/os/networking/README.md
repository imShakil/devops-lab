# Networking Fundamentals

Networking fundamentals enable reliable service communication and production debugging.

## Core Concepts

- OSI/TCP-IP model
- IP addressing and subnetting
- DNS resolution flow
- Routing and NAT
- TCP vs UDP behavior

## Essential Commands

```bash
ip a
ip route
ss -tulpen
dig example.com
curl -v https://example.com
traceroute example.com
```

## Best Practices

- Use clear network segmentation.
- Monitor latency, packet loss, and throughput.
- Standardize DNS and firewall rules.
